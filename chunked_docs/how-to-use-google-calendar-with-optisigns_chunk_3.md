# How to use Google Calendar with OptiSigns

## What You'll Need

To use the Google Calendar app, you'll need:

* An OptiSigns [**Standard plan or higher**](https://www.optisigns.com/pricing)
* A Google account with Calendar
* An [**OptiSigns-capable device**](https://support.optisigns.com/hc/en-us/articles/360021855653-What-hardware-and-devices-are-supported)
* A screen, [**set up and paired**](https://support.optisigns.com/hc/en-us/articles/18823504383891-OptiSigns-Getting-Started-Guide)

|  |
| --- |
| **NOTE** |
| In order to display a Google Calendar, you'll need to grant OptiSigns certain permissions. See our [**Privacy Policy**](https://www.optisigns.com/privacy-policy)for what we do with those permissions. |
